Module Main


module Main: sig .. end
Module destine a la creation et representation de main de 5 cartes
Author(s): : casier_s;duques_p Univ Lille 1

le type de donnees



Une main est un ensemble de cinq cartes differentes d'un jeu de poker.
type main

Constructeur et selecteur


val cree : Carte.carte * Carte.carte * Carte.carte * Carte.carte * Carte.carte ->
       main

cree (c1,c2,c3,c4,c5) cree une main avec les cinq cartes donnees.
val carte : main -> int -> Carte.carte
carte m i renvoie la carte de rang i dans l'ordre croissant dans la main m. Les cartes carte m 1, carte m 2, carte m 3, carte m 4, carte m 5 sont les cinq cartes de la main m et leurs hauteurs sont dans l'ordre croissant. CU : l'entier i doit etre compris entre 1 et 5. Sinon, declenche l'exception Failure "Main.carte : indice incorrect".

Procedure dimpression


val imprimer : main -> unit
imprimer m imprime la main m. Par exemple, l'impression a la forme 2C, 3K, 8K, 9K, RP

Divers


val main_alea : unit -> main
main_alea () renvoie une main de cinq cartes choisies aleatoirement dans un jeu de 52 cartes.
val distribue : int -> main array
distribue n renvoie un tableau de n mains extraites d'un jeu de 52 cartes. CU : l'entier n doit etre compris entre 2 et 5. Sinon, declenche l'exception Failure "Main.distribue : nombre joueurs incorrect".
